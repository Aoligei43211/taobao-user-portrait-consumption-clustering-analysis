# 淘宝用户画像消费聚类分析

## 项目简介

本项目基于阿里云天池 1000 条淘宝用户消费数据，采用三种聚类算法（KMeans、KPrototypes、KModes）对用户进行分群画像，通过 Streamlit + Plotly 构建交互式可视化报表，最终结合业务场景给出差异化的首页推荐策略。

**数据字段**：用户ID、用户姓名、商品ID、商品名称、商品类别、单价、购买时间、购买数量、消费金额、用户城市、用户性别、用户年龄

**控制变量**：商品类别、消费金额、用户性别、age（年龄分档）

---

## 三种聚类方法横向对比

| 维度 | KMeans | KPrototypes | KModes |
|------|--------|-------------|--------|
| **数据类型** | 纯数值 | 混合（数值+类别） | 纯类别 |
| **特征处理** | 独热编码 + 标准化 | log变换 + 标准化 | 分位数分箱 |
| **距离度量** | 欧氏距离 | 欧氏距离 + 匹配距离 | 汉明距离（匹配距离） |
| **优点** | 计算快、可解释性强、指标成熟 | 保留类别信息、无需编码膨胀、适合混合数据 | 无需数值化、抗异常值、结果直观 |
| **缺点** | 独热编码导致维度灾难、对异常值敏感 | 计算复杂度高、γ权重难调 | 信息损失大（分箱粒度粗）、簇中心不唯一 |
| **适用场景** | 数值型数据为主、追求计算效率 | 混合数据、需要同时处理数值和类别特征 | 全类别数据、探索性分析 |
| **轮廓系数** | 可直接使用 | 需提取数值部分计算 | 需将类别编码为数值后计算 |
| **最佳簇数** | 由肘部法则/轮廓系数/CH指数决定 | 由肘部法则/轮廓系数/CH指数决定 | 由肘部法则/轮廓系数/CH指数决定 |

## 方法选择与最佳画像

### 为什么选择 KPrototypes？

1. **数据匹配度最高**：原始数据包含数值（消费金额）和类别（商品类别、性别、年龄）混合特征，KPrototypes 天然支持混合数据，无需像 KMeans 那样独热编码导致维度膨胀，也无需像 KModes 那样强行分箱损失信息。

2. **业务可解释性最强**：聚类中心直接给出每个簇的"典型用户"特征（如：美妆-女-青年-中等消费），便于业务人员理解。

3. **指标表现均衡**：在轮廓系数和 CH 指数上表现稳定，不会出现 KMeans 因距离度量不适配类别特征而导致的偏差。

### 最优画像识别

通过 KPrototypes 聚类分析，识别出具有显著业务意义的用户群体。以下是一个典型案例：

**被抛弃的画像：男-老年-美妆**

该群体占比仅约 1.8%，在聚类中表现为：
- 消费金额偏低
- 商品类别高度集中在美妆
- 年龄与性别的组合与主流美妆消费群体（女-青年/中年）差异巨大

**业务判断**：该群体样本量过小，不具备独立运营价值，建议将其合并到相邻簇中，或在推荐策略中作为长尾用户处理。

## 差异化首页推荐策略

基于 KPrototypes 聚类结果，结合淘宝商品种类齐全的特点，为不同用户群体设计差异化首页推荐：

### 策略一：高消费电子产品偏好簇（男-中年/老年-电子产品-高消费）
- **首页推荐**：最新款电子产品首发、数码配件套装、以旧换新专区
- **运营动作**：推送高端品牌旗舰店优惠券、提供分期付款选项
- **商品池**：笔记本电脑、智能手表、显示器、蓝牙耳机

### 策略二：美妆主力消费簇（女-青年/中年-美妆-中高消费）
- **首页推荐**： seasonal 新品美妆、明星同款、直播带货专区
- **运营动作**：会员积分翻倍、美妆套装组合优惠、KOL 种草内容
- **商品池**：粉底液、防晒霜、口红、护肤品套装

### 策略三：性价比家居簇（不限性别-中年/老年-家居-中低消费）
- **首页推荐**：家居用品促销、日用百货特价、团购拼单
- **运营动作**：满减活动、家庭套装推荐、节假日主题促销
- **商品池**：餐桌、沙发、四件套、洗发水

### 策略四：年轻运动潮流簇（不限性别-青年-运动/服装-中消费）
- **首页推荐**：运动装备上新、潮牌联名款、健身季卡套餐
- **运营动作**：学生认证优惠、社交分享返现、运动打卡活动
- **商品池**：跑步鞋、瑜伽垫、篮球、T恤、羽绒服

### 策略五：文化教育簇（不限性别-青年/中年-书籍-低中消费）
- **首页推荐**：新书推荐、电子书套餐、知识付费课程
- **运营动作**：读书打卡奖励、作者签售活动、会员借阅服务
- **商品池**：心理学、时尚杂志、教材教辅

### 策略六：长尾用户（男-老年-美妆等低占比群体）
- **首页推荐**：通用热销商品、平台级促销活动
- **运营动作**：不单独运营，纳入全量推荐池，通过协同过滤推荐
- **商品池**：全品类 Top 100 热销

## 运行方式

### 本地运行
```bash
# 安装依赖
pip install -r requirements.txt

# 运行任一应用
streamlit run kmeans_app.py
streamlit run kprototypes_app.py
streamlit run kmodes_app.py
```

### Streamlit Cloud 部署
1. Fork 本仓库
2. 登录 [Streamlit Cloud](https://share.streamlit.io)
3. 点击 "New app"，选择此仓库
4. 设置 Main file path 为 `kprototypes_app.py`（或其他）
5. 点击 "Deploy"

## 技术栈

- **数据处理**：pandas, numpy

- **可视化**：plotly, streamlit
